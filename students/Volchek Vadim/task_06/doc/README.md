## Тема
Стратегия тестирования: unit, integration, e2e.

## Test Pyramid
- Unit: инварианты домена и обработчики команд.
- Integration: API + репозиторий.
- E2E: полный пользовательский сценарий.

## Что проверяется
1. Создание бага.
2. Назначение бага на группу.
3. Негативный кейс для невалидного приоритета.
4. Сквозной поток create -> assign -> get.
