## Тема
CQRS и read models: разделение записи и чтения.

## Реализовано
- Write model для изменения состояния бага.
- Read model для быстрого чтения списка багов по проекту/статусу.
- Projection-обновление read model из событий.

## События
- `BugCreated`
- `BugAssigned`
- `BugStatusChanged`
