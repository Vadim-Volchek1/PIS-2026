## Тема
Микросервисы и bounded contexts.

## Bounded Contexts
- `request-service`: создание и изменение багов.
- `group-service`: управление командами/группами.
- `notification-service`: уведомления о новых и назначенных багах.

## Интеграция
- Асинхронные события через RabbitMQ.
- API Gateway для внешних запросов.
- Базовая отказоустойчивость через retry/circuit breaker на клиенте.
