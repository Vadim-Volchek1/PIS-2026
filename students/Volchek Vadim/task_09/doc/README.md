## Тема
Protocol Buffers и gRPC для межсервисного взаимодействия.

## Реализовано
- Контракт `proto/bug_service.proto`.
- Заглушка gRPC-сервера (`grpc/server.py`) с методами:
  - `CreateBug`
  - `GetBug`
  - `ListActiveBugs` (server streaming)
- Пример gRPC-клиента (`grpc/client.py`).

## Версионирование
Для расширения протокола используются новые поля с новыми номерами, существующие номера не переиспользуются.
